# 내부 구축 플랫폼 모듈

이 문서는 ISMS-P 대응 과정에서 내부에 구축해야 할 플랫폼 SaaS 모듈을 정리한 자료입니다. 내부 구축 대상은 조직의 실제 운영 데이터와 심사 증적을 함께 설명해야 하는 영역입니다. 외부 제품이 탐지와 수집을 맡더라도, 우리 서비스의 처리 기준과 예외 승인을 설명하는 최종 증적 레이어는 내부에 두는 편이 안전합니다.

상위 문서: [ISMS-P 클라우드 SaaS 기술 준비 런북](./isms-p-cloud-saas-runbook.md)
